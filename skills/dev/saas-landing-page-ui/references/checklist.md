# SaaS Landing Page UI Checklist

Use this checklist to classify a page and decide the next improvements. Default target for production pages: **Level 4**.

## Scoring

Score each category from 1 to 4:

- **1:** Generic/template; works but feels basic.
- **2:** Intentional; improved identity but still shallow.
- **3:** Cohesive; product-led and clearly designed.
- **4:** Crafted; content-specific, polished, and conversion-oriented.

The page level is the lowest repeated score across core categories. A page with Level 4 visuals but Level 2 copy should be treated as Level 2–3 overall.

## 1. Page Job and Conversion

- [ ] Target audience is clear.
- [ ] Primary conversion is clear.
- [ ] Main CTA appears in the header and hero with consistent labeling when destination is the same.
- [ ] Secondary CTA supports exploration without competing with the primary CTA.
- [ ] Each section answers a likely visitor question or objection.

## 2. Layout and Flow

- [ ] The page avoids repeating the same section pattern too many times.
- [ ] Section density varies intentionally.
- [ ] Hero, proof, feature, social-proof, and CTA sections flow naturally.
- [ ] Content feels designed for the available space.
- [ ] Responsive behavior preserves hierarchy and rhythm.

## 3. Product Visuals

- [ ] Visuals are product-specific, not generic stock imagery.
- [ ] Each visual demonstrates a concrete product value.
- [ ] Dashboard/app screenshots are cropped or composed around the key idea.
- [ ] Product visuals carry enough narrative to let copy become shorter.
- [ ] Visual treatment is consistent with the rest of the brand system.

## 4. Typography and Hierarchy

- [ ] Hero headline has clear dominance.
- [ ] Subcopy is readable and not visually equal to the headline.
- [ ] Widths and line breaks guide the eye into the next content.
- [ ] Section headings, labels, cards, and CTAs have distinct roles.
- [ ] Spacing supports scanning without making the page feel empty.

## 5. Copywriting

- [ ] Headlines are concise and specific.
- [ ] Copy emphasizes customer outcomes, not just product mechanics.
- [ ] Paragraphs are short enough to scan.
- [ ] Product names are not repeated unnecessarily.
- [ ] Feature descriptions answer why the feature matters.

## 6. Navigation and Mega Menus

- [ ] Header is visually balanced.
- [ ] Important actions have stronger hierarchy than passive links.
- [ ] Product/resource links reveal useful depth when appropriate.
- [ ] Mega menus group information around visitor intent.
- [ ] Open/close and hover states feel continuous, not abrupt.

## 7. Color and Brand

- [ ] Accent color has a clear role.
- [ ] Color is threaded through visuals, highlights, and interactive states.
- [ ] Neutral surfaces allow CTAs and product visuals to stand out.
- [ ] Gradients/glows/noise are used sparingly and consistently.
- [ ] Contrast is accessible.

## 8. Motion and Interaction

- [ ] Motion helps orientation or reveals useful context.
- [ ] Hover states add polish without hiding essential content.
- [ ] Menus and tabs preserve spatial continuity.
- [ ] Load/scroll animations are subtle and do not delay comprehension.
- [ ] Reduced-motion handling exists for production implementation.

## 9. Detail Pass

- [ ] Alignment is consistent across sections.
- [ ] Border radius, shadows, strokes, and dividers use a coherent system.
- [ ] Icon style and visual density are consistent.
- [ ] Cards and product UI have enough internal spacing.
- [ ] The final page screenshot still reads clearly when zoomed out.

## Level Diagnosis

### If the page feels Level 1

Fix first:

1. Replace stock/generic images with product visuals.
2. Create a stronger hero hierarchy.
3. Shorten copy.
4. Make CTAs consistent.
5. Vary section layouts.

### If the page feels Level 2

Fix first:

1. Curate product visuals instead of showing full dashboards.
2. Add stronger section rhythm.
3. Improve nav hierarchy and important dropdowns.
4. Make color usage more deliberate.
5. Improve copy from descriptive to outcome-led.

### If the page feels Level 3

Fix first:

1. Balance typography and spacing.
2. Refine motion continuity.
3. Add interaction details that reveal context.
4. Thread color through the entire page.
5. Make every section feel custom-fit to its content.

### If the page is targeting Level 4

Verify:

1. Every section has a clear conversion purpose.
2. Product visuals show the exact value being claimed.
3. Copy states how the product helps the customer.
4. Motion and hover states feel premium but restrained.
5. A zoomed-out screenshot has rhythm, balance, and a clear visual path.
